# docs/sources -- Derivative transcripts

This directory contains text-only transcripts and translations derived from
historical xwpe documentation sources. The raw originals are preserved in
the companion repository [xwpe-archive](https://codeberg.org/mendezr/xwpe-archive).

Each file here is our own work (transcription, translation, or summary) and
is licensed under GPL-2, the same as the rest of xwpe.

## Files

| File | Source | Description |
|------|--------|-------------|
| `identicalsoftware-maillist.md` | identicalsoftware.com mailing list | Plain-text thread index (subjects, authors, dates) |
| `linux-cz-1998-06.md` | linux.cz 1998 Czech tutorial | English translation of the Czech xwpe tutorial |
| `lug-s-debugger-course.md` | lug-s.org German debugger course | Transcription of GIF images + English translation |
